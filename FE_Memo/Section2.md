# 2進数

```
(例2-1)
	64kビット/秒の回線を用いて10^6のファイルを送信するとき、伝送におよそ何秒掛かるか。  
	ここで、回線の伝送効率は80％とする。

　----------------------------------------------------------------------------------

	回線速度が64kビット／秒で伝送効率が80%なので、

	　64kビット／秒＝8kバイト／秒
	　8k×0.8＝6.4kバイト／秒

	1秒間の伝送可能量は最大で6.4kバイトと計算できます。
	したがって106バイトのファイルを送信するときの伝送時間は、

	　1,000,000÷6,400＝156.25(秒)

```
